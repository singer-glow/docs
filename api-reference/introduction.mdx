## Authentication

All API requests require authentication using a Bearer token.

```bash
curl -X GET "https://api.glowsecurity.com/v1/endpoint" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Rate Limits

API requests are rate limited. Contact your Glow representative for details on your plan's limits.
